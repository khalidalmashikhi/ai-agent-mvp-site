# AI Agent-MVP Website

Premium bilingual landing page for **AI Agent-MVP** / **النموذج الأولي لتطوير AI Agent-MVP**.

## Run locally

```bash
npm install
npm run dev
```

Open: http://localhost:3000

## Deploy

Upload this repository to GitHub and connect it to Vercel or Netlify.

## Notes

- Built with Next.js + TypeScript + CSS.
- Arabic/English switch is included.
- Dark premium visual direction.
- No visual icons in content sections; only a custom abstract logo mark.
